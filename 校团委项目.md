# 站群迷惑行为

（官方文档对于各种问题和站群的操作说了和没说没啥区别，而且有些地方说的不对）

## 迷惑行为
bug 贼多，一些常见 bug 的解决和神奇的功能的解释

1. 备份
没有任何用处，备份的是你上次的 htm 等等文件夹，不会对你的拖拽的东西进行备份

2. 可视化拖拽页面
- 火狐无法拖拽，请使用 chrome
- ie 会非常卡不要用
- 有时候拖着拖着就不能拖了，请刷新页面重新来
- 拖进去的时候会有默认的样式显示，选择左边的栏目是将内容和信息绑定
- 可视化拖拽页面有时会和略览页面显示不同（对于某些东西显示不出来），请以略览为准
- 当拖入空的组件时会导致可以拖拽的窗口不可见，请f12修改一下加个内容让窗口被撑大

3. frag 的书写

- 必须为窗口xx(xx为数字)，当然还可以是面板xx不过好像没啥用
- 窗口xx不允许存在嵌套

4.灵活运用组件
4.1 带图片的超链接
单发一篇文章，文章里上传一张图片然后点源码在外面嵌一层a标签即可，用新闻内容这个组件拖

4.2 带文字的超链接
同上

4.3 单显示一张图片
法1：发一篇只有一个图片的文章，用新闻内容这个组件拖
法2：修改图文标题的样式，在拖进去的高级配置里修改

4.3 导航自定义样式
在模板->样式菜单->导航管理里 新增样式即可
注：拖导航菜单的时候也可以加，但加了就不能改了(鬼畜bug)

4.4 文章内容是引用外部的网站
发文章的时候选择一个链接，输入指定的网址就行，这样点击就会跳过去

4.5 日常容易页面崩溃

4.6 多人修改同一页面可能会出现问题

4.7 导航里面的组织结构修改为链接到文章的详情页,导新闻列表模块并且适当修改样式

4.8 修改高级样式最优先，能强制覆盖选择强制覆盖，最差情况下用js动态删除节点

4.9 千万不要乱点，死的很惨的
